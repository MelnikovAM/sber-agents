# Workflow по тасклисту

- Выполняй задачи строго по очереди согласно docs/tasklist.md.
- Перед каждой задачей:
  - Согласуй с автором предполагаемое решение (код-отрывки по ключевым частям).
  - Дождись подтверждения.
- После подтверждения:
  - Реализуй задачу полностью.
  - Ожидай проверки и подтверждения выполнения.
- После подтверждения:
  - Отметь задачу выполненной в tasklist.md.
  - Обнови блок прогресса (таблица, чекбокс).
  - Согласуй переход к следующей задаче.
- В конце каждой задачи делай коммит с осмысленным сообщением.

Придерживайся принципа KISS. Никаких лишних действий и лишнего кода.
